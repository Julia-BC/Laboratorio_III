## 📄 Sobre o projeto
O **Agenda Beauty** é uma plataforma desenvolvida para **organizar e gerenciar agendamentos de serviços em salões de beleza**.

O sistema permite cadastrar clientes, funcionários e serviços, além de controlar horários disponíveis para agendamento.

Inicialmente, o projeto também incluía a ideia de integrar **automação com chatbot**, porém essa funcionalidade não foi implementada na versão final do projeto acadêmico.

O foco principal foi a **estrutura do sistema**, **modelagem do banco de dados** e o desenvolvimento das funcionalidades essenciais de agendamento.

## 🎯 Objetivo principal
Aplicar na prática conhecimentos de:
- Modelagem de banco de dados
- Desenvolvimento web com framework
- Estruturação de sistemas
- Organização de código
- Integração entre front-end e back-end

Projeto desenvolvido como **atividade acadêmica**.

## 🧱 Funcionalidades do sistema
- Cadastro de clientes
- Cadastro de salões
- Cadastro de funcionários
- Gerenciamento de serviços
- Agendamento de horários
- Visualização da agenda
- Controle de dados do estabelecimento

## 👥 Tipos de usuários do sistema
O sistema foi projetado para três perfis principais:

**Cliente**
- Realiza agendamentos
- Visualiza horários disponíveis

**Funcionário**
- Responsável pelos atendimentos
- Vinculado a serviços específicos

**Gerente / Dono do salão**
- Gerencia o estabelecimento
- Controla serviços, funcionários e agenda

## 🏗️ Arquitetura do sistema
O projeto foi desenvolvido utilizando o padrão:

**MVC (Model - View - Controller)**

Estrutura utilizada pelo Laravel para separar responsabilidades:

- **Model** → comunicação com o banco de dados  
- **View** → interface do sistema  
- **Controller** → regras de negócio  

---

## 🗄️ Modelagem do banco de dados
O banco de dados foi estruturado para gerenciar as seguintes entidades principais:

- Cliente
- Empresa (Salão)
- Funcionário
- Serviços
- Agendamentos
- Endereço

Relacionamentos entre as tabelas garantem a integridade dos dados e organização do sistema.

## 🚀 Como visualizar o projeto
1. Clone o repositório:
git clone https://github.com/SEU-USUARIO/agenda-beauty.git

2. Acesse a pasta do projeto:
cd agenda-beauty

3. Instale as dependências do Laravel:
composer install

4. Configure o arquivo .env
   
5. Execute o projeto:
php artisan serve
